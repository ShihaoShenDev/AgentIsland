本项目基于 .NET 8 + Avalonia UI 开发 ClassIsland 桌面插件，构建与打包围绕以下体系展开：

1. 项目文件与依赖
- AgentIsland.csproj 指定 TargetFramework 为 net8.0-windows，使用 ImplicitUsings/Nullable 现代 C# 特性。
- 通过属性 ClassIslandPluginSdkVersion 集中管理 ClassIsland.PluginSdk 版本（当前 1.7.106.2-dev-v2），避免散落的版本号。
- 关键运行时依赖 DotNetCampus.ModelContextProtocol、AgentClientProtocol、Sentry 以 NuGet 引用；同时通过 <None Include="..." CopyToOutputDirectory="PreserveNewest"> 将 ModelContextProtocol/System.Text.Json/System.IO.Pipelines/Sentry 等 DLL 显式复制到输出目录，确保 ClassIsland 宿主能直接加载这些非托管/特殊依赖。
- manifest.yml 声明插件元数据（id/name/version/apiVersion/supportedOSPlatforms 等），被 publish 流程读取并嵌入 cipx 包。

2. 本地构建脚本（PowerShell）
- build-debug.ps1：调用 dotnet build 生成 Debug 输出，然后启动宿主 ClassIsland.Desktop.exe（路径来自环境变量 ClassIsland_DebugBinaryDirectory），并以 -epp 参数指向 bin\Debug\net8.0-windows 作为扩展插件目录，便于热调试。
- build-release.ps1：dotnet publish -c Release 发布到 bin\Release\net8.0-windows\publish，再以相同方式注入宿主运行。
- create-cipx.ps1：dotnet publish -c Release -p:CreateCipx=true 触发 ClassIsland Plugin SDK 的内置打包目标，最终产出 cipx 目录下的 .cipx 压缩包。
- 三个脚本均先 taskkill /im ClassIsland.Desktop.exe /f 杀掉可能残留的宿主进程，避免端口/文件占用冲突。

3. CI 流水线（GitHub Actions）
- .github/workflows/cipx.yml 在 windows-latest 上安装 dotnet 8.0.x，执行 dotnet publish -c Release -p:CreateCipx=true，并将 cipx 目录作为 Artifact 上传，实现“push/pull_request/main 分支”自动构建插件包。

4. 产物结构约定
- cipx/ 目录存放最终可发布的 AgentIsland.cipx 及 checksums.md。
- artifacts/verify-acp 与 artifacts/verify-final 是历史验证用的发布快照（含 deps.json/dll/icon.png/manifest.yml），供离线校验使用。
- 根目录 icon.png、README.md、manifest.yml 通过 <None Update="..." CopyToOutputDirectory="PreserveNewest"> 随编译一并拷贝，参与 cipx 打包。

5. 开发者应遵循的规则
- 新增 NuGet 依赖时，若该依赖需要由 ClassIsland 宿主直接加载（如 ModelContextProtocol/System.Text.Json 等），需在 csproj 中追加对应的 <None Include="$(NuGetPackageRoot)..." CopyToOutputDirectory="PreserveNewest"> 条目。
- 修改插件元数据（id/name/version/apiVersion 等）统一编辑 manifest.yml，不要分散在各处。
- 本地调试优先使用 build-debug.ps1，正式发布或打 cipx 包时使用 build-release.ps1 / create-cipx.ps1，避免手动拼 dotnet 命令导致输出目录不一致。
- 提交前确保 cipx/ 下产物与最新源码一致，CI 会在 push/pull_request 时重新构建并上传 Artifact。
