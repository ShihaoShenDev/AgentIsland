模块按职责分为两个扁平子目录：
- `Models/`：承载所有可序列化与 MVVM 绑定的数据类型。其中 `AgentIslandSettings` 是根设置对象，聚合 `ObservableCollection<AcpAgentProfile>` 与 `ObservableCollection<AiTextEntry>`，并在集合变更时通过 Hook/Unhook 模式维护派生属性（如 `TotalAgentCount`、`ConnectionAddress`、遥测开关状态）；`AiTextComponentSettings` 继承自 `ObservableRecipient`，作为组件级轻量配置；`McpTransportMode` 枚举区分 StreamableHttp 与 SSE 两种传输协议；`ToolResults.cs` 集中存放与 ClassIsland 课程表交互的不可变记录类型（`CurrentClassResult`、`ScheduleResult` 等），并通过 `AgentIslandJsonContext` 上的 `[JsonSerializable]` 注解启用 System.Text.Json 源生成器以支持驼峰命名序列化。
- `Helpers/UiThreadHelper.cs`：仅暴露 `RunOnUi<T>` / `RunOnUi(Action)` 两个静态方法，基于 Avalonia `Dispatcher.UIThread.CheckAccess` 做无锁切换，避免跨线程调用 UI 资源。
依赖方向单向：Helpers 不引用 Models；Models 之间通过 `ObservableObject` / `ObservableRecipient` 松耦合，仅 `AgentIslandSettings` 反向订阅其子集合项的属性变更。
