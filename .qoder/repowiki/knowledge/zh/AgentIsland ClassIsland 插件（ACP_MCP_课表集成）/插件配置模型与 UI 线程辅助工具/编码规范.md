- 配置类统一继承 `CommunityToolkit.Mvvm.ComponentModel.ObservableObject`，字段使用私有 backing field + `[JsonPropertyName]` 标注 JSON 键名，并通过 `SetProperty(ref _field, value)` 赋值触发通知。
- 简单只读字段采用 `[ObservableProperty]` 部分类语法自动生成 getter/setter 与通知，减少样板代码。
- 对 `ObservableCollection<T>` 属性在 setter 中先 Unhook 旧集合再 Hook 新集合，确保集合增删与元素属性变更都能正确传播到派生属性。
- 对外暴露的 DTO 结果类型优先使用 `sealed record` 不可变结构，配合 `AgentIslandJsonContext` 的 `[JsonSerializable]` 注解进行源生成序列化。
