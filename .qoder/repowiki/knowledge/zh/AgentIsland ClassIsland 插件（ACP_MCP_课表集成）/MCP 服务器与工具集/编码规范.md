- 工具方法通过 `[McpServerTool(Name=..., ReadOnly=..., Structured=...) ]` 属性声明为 MCP 工具，并将核心逻辑下沉到同名 `*Core()` 私有方法以便复用。
- 每个工具入口先通过 `IAppHost.GetService<SentryTelemetryService>()` 获取遥测，并用 `WithInstrumentation(name, core)` 包裹实际执行，失败时回退直接调用。
- 对 UI 状态的读写一律封装在 `UiThreadHelper.RunOnUi(() => ...)` 或 `Dispatcher.UIThread.InvokeAsync` 中，避免跨线程访问 Avalonia 控件。
- 日志记录遵循 `LogDebug("调用 <tool_name>")` 进入 + `LogInformation("业务语义: {参数}")` 关键路径的双层粒度。
- 需要手动定义输入参数的工具实现 `IMcpServerTool`，并以静态 `JsonElement` 字段内联 JSON Schema，配合 `ReadRequiredString`/`ReadOptional*` 解析器读取 `context.InputJsonArguments`。
