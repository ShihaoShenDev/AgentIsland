- UI 控件通过继承 `ClassIsland.Core.Abstractions.Controls` 下的泛型基类（`ComponentBase<T>` / `SettingsPageBase` / `ActionSettingsControlBase<T>`）获得 Settings 注入与生命周期钩子。
- 设置页与组件使用特性声明元信息：`[ComponentInfo(...)]` 用于运行时组件注册，`[SettingsPageInfo(id, name, category: SettingsPageCategory.External)]` 用于设置窗口导航。
- 视图构造或 `OnInitialized` 中直接将 `DataContext = Plugin.Settings`，XAML 通过绑定访问 `AcpAgents`、`AiTextEntries` 等集合属性，避免在后台代码中手动同步。
- 对 `Plugin.Settings` 下可变集合的订阅遵循 Loaded/Unloaded 配对模式：在 Loaded 中订阅 `CollectionChanged`/`PropertyChanged`，在 Unloaded 中移除对应事件处理器，防止内存泄漏。
- 按钮操作通过 `sender is Button { Tag: T }` 模式从 Tag 取回当前行模型，再调用 `Plugin.Settings.XxxList.Add/Rmoeve` 修改集合。
