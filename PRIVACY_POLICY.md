# AgentIsland 遥测隐私政策

**生效日期：** 2026年6月27日  
**最后更新：** 2026年6月27日

---

## 一、概述

AgentIsland（以下简称"本插件"）是一个运行于 ClassIsland 桌面客户端的插件。为持续改进插件质量和稳定性，本插件集成了由 [Sentry](https://sentry.io/) 提供的遥测服务。

本政策旨在向您说明本插件收集哪些数据、如何使用这些数据、以及您的权利。

---

## 二、收集的数据类型

### 2.1 我们收集的数据

| 数据类型 | 说明 | 示例 |
|---------|------|------|
| **未处理异常** | 插件运行过程中发生的崩溃和错误信息 | 堆栈跟踪、错误消息 |
| **性能指标** | MCP 工具调用的延迟和成功率 | 工具调用耗时、请求状态 |
| **生命周期事件** | 插件启动、停止、MCP 服务器连接等关键事件 | 面包屑记录（Breadcrumb） |
| **插件元数据** | 插件版本号、运行环境标识 | `plugin=AgentIsland` 标签 |

### 2.2 我们不收集的数据

| 数据类型 | 说明 |
|---------|------|
| **个人课程信息** | 不收集课表内容、科目名称、教师信息等 |
| **个人身份信息** | 不收集 IP 地址、主机名、用户名等 PII 数据 |
| **用户输入内容** | 不收集用户在设置中输入的自定义内容 |
| **敏感数据** | 不收集任何密码、密钥、令牌等凭证信息 |

---

## 三、数据使用目的

收集的数据仅用于以下目的：

1. **错误诊断与修复** — 识别和修复插件中的缺陷
2. **性能优化** — 分析工具调用延迟，优化响应速度
3. **稳定性监控** — 追踪崩溃率，提升整体可靠性
4. **功能改进** — 了解功能使用情况，指导后续开发方向

---

## 四、数据处理方

本插件使用 **Sentry**（由 Functional Software, Inc. 运营）作为遥测数据处理器。

- **Sentry 官网：** https://sentry.io/
- **Sentry 隐私政策：** https://sentry.io/privacy/
- **Sentry 安全白皮书：** https://sentry.io/security/

Sentry 作为独立的数据处理方，按照其自身隐私政策处理所接收的数据。

## 六、用户控制与同意机制

### 6.1 同意流程

1. 本插件安装后，遥测功能**默认处于开启状态**
2. 但遥测**不会立即生效**，需要您主动同意本隐私政策
3. 在 ClassIsland 设置中找到 **AgentIsland / 遥测与隐私** 页面
4. 点击"同意"按钮并确认后方可启用遥测

### 6.2 撤回同意

您可以随时通过以下方式撤回同意：

1. 进入 **AgentIsland / 遥测与隐私** 设置页面
2. 点击"撤回同意"按钮并确认
3. 撤回后，Sentry SDK 将立即关闭，不再收集任何新数据

> **注意：** 已上报至 Sentry 的历史数据无法撤回，但撤回后不会再产生新的数据。

### 6.3 遥测开关

除隐私协议同意外，您还可以使用遥测开关完全禁用遥测：

- 关闭"启用遥测数据收集"开关后，无论是否已同意隐私政策，遥测都将停止
- 重新开启开关后，仍需已同意隐私政策才会恢复遥测

### 6.4 自定义 DSN 豁免

如果您填写了自定义 Sentry DSN 地址：

- 遥测数据将直接发送至您自己的 Sentry 项目
- 此时**隐私协议同意检查将被跳过**，因为您已自行掌控数据流向
- 您可以随时清空自定义 DSN 恢复默认模式

## 七、数据保留期限

| 数据类型 | 保留期限 |
|---------|---------|
| Sentry 事件数据 | 默认 90 天（可由项目管理员调整） |
| Sentry 面包屑数据 | 随事件数据一同保留 |
| 本地缓存数据 | 最多 30 条离线事件，发送后自动清理 |
