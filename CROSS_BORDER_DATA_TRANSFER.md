# AgentIsland 跨境数据传输协议

**生效日期：** 2026年6月27日  
**最后更新：** 2026年6月27日

---

## 一、概述

本协议是 [AgentIsland 遥测隐私政策](PRIVACY_POLICY.md) 的补充文件，专门说明遥测数据在跨境传输方面的安排。

在同意隐私政策之前，您需要先阅读并同意本协议。

---

## 二、数据流向

本插件使用 **Sentry**（由 Functional Software, Inc. 运营）作为遥测数据处理方。

Sentry 的服务器位于**美国**及其他国家和地区。当您启用遥测功能后，技术数据将通过互联网从您的设备传输至 Sentry 的服务器。

### 数据流向示意

```
您的设备（中国）
    │
    │ HTTPS / TLS 加密传输
    ▼
Sentry 服务器（美国）
    │
    │ 存储与处理
    ▼
Sentry 数据中心（美国）
```

---

## 三、传输的数据内容

跨境传输的数据**仅包含**以下技术信息：

| 数据类型 | 说明 |
|---------|------|
| 异常堆栈跟踪 | 插件崩溃时的错误信息和调用栈 |
| 性能指标 | MCP 工具调用延迟、成功率等聚合数据 |
| 面包屑事件 | 插件生命周期事件记录（如启动、连接等） |
| 插件标签 | `plugin=AgentIsland` 标识 |

跨境传输的数据**不包含**：

- 个人课程信息或课表内容
- IP 地址、主机名等个人身份信息（已在本地通过 `SendDefaultPii = false` 过滤）
- 用户输入的任何自定义内容
- 密码、密钥、令牌等凭证信息

---

## 四、法律基础

根据《中华人民共和国个人信息保护法》（PIPL）第三十八条，向境外提供个人信息需满足以下条件之一。本插件采取以下方式合规：

- **取得个人的单独同意** — 通过设置页面中的"同意"按钮获取您的明确授权
- 您有权随时撤回同意，撤回后数据将不再继续传输

---

## 五、安全保障措施

我们采取以下措施保障跨境传输数据的安全：

### 5.1 数据最小化

- 仅传输必要的技术诊断数据
- 不传输任何可识别个人身份的信息
- 不传输用户生成的内容或课程数据

### 5.2 传输安全

- 所有数据传输均通过 **HTTPS/TLS** 加密
- 传输链路端到端加密，防止中间人攻击

### 5.3 存储安全

- Sentry 对静态数据进行加密存储
- Sentry 默认保留事件数据 **90 天**，之后自动永久删除
- 支持自定义数据保留策略

### 5.4 访问控制

- 仅插件开发者可访问 Sentry 项目中的遥测数据
- Sentry 提供完整的审计日志

## 六、您的权利

根据适用的数据保护法律，您享有以下权利：

| 权利 | 说明 | 如何行使 |
|------|------|---------|
| **知情权** | 了解数据如何被跨境传输和处理 | 阅读本文件及隐私政策 |
| **同意权** | 在数据传输前获得您的明确同意 | 在设置页面点击"同意" |
| **撤回权** | 随时撤回同意，停止数据传输 | 在设置页面点击"撤回同意" |
| **访问权** | 请求访问已传输的关于您的数据 | 通过 GitHub Issues 联系开发者 |
| **删除权** | 请求删除已传输的数据 | 通过 GitHub Issues 联系开发者 |
